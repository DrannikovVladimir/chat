## Simple analog of [Slack (Chat)](https://stark-savannah-34514.herokuapp.com/)   

### Requirements

* [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli)   

### Libraries used  

```
* React   
* Redux Toolkit   
* Formik   
* React Router Dom   
* React Bootstrap   
* Soket.io   
* i18next
```   

### Install   
```sh
$ make install
```   
   

### Run   
```sh
$ make start   
open http://localhost:5000
```
